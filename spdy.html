What is it?

SPDY is an experiment with protocols for the web by Google started over 3 years ago.

It is the basis of HTTP/2.0.

Its goal is to reduce the latency of web pages.

It is being used by

  • Google Chrome & All Google Web Properties
  • Mozilla Firefox
  • Twitter
  • Amazon Silk
  • Others: Cotendo, Strangeloop, iPhone client, Apache mod-spdy, nginx, jetty, netty, libraries in python, node.js, erlang, ruby, go, and C

  • Exclusively client-initiated requests
  • Uncompressed request and response headers
